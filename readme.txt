== Description ==

Adds the Twitter @Anywhere JavaScript code to your blog, enabling a few of the @Anywhere features: Hovercards and Auto-linkification of @usernames.

In order to use @Anywhere, you must first register your blog for a free API key with Twitter. You can do so at: http://dev.twitter.com/apps

== Installation ==

1. Upload `AddTwitterAnywhere.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter your Twitter @Anywhere API key on the 'Settings' > 'Add Twitter @Anywhere' settings page
